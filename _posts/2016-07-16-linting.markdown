jshint
======
 - install
 {% highlight shell%}
    sudo npm install -g jshint
 {% endhighlight %}

 - configuration
**enforcing** is making the `jshint` more strict, **relaxing** is to suppress some warning.
[.jshintrc](https://github.com/jshint/jshint/blob/master/examples/.jshintrc) option explanation.
